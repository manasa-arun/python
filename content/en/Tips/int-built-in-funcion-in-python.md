## int():
Python `int()` function converts the specified value into an integer number, 
that means if the number is in float,decimal numbers can convert in to integers with this `int()` function.

**Example**:
```
x = 12.5
y = 3.9

a= int(x)
b =int(y)

print(x,y)
```
***output:**
```
12 3
```
