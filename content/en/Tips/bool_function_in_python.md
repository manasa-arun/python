## bool():
`bool()` function is a built in functions in python.

Python `bool()` function is used to return or convert a value to a Boolean value 

i.e., True or False, using the standard truth testing procedure.

**Example:**
```
bool([])
```
**output**
```
false
```

**Example:**
```
bool([none])
```
**output**
```
true
```
